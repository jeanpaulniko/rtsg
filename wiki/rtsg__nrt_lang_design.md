# NRT-Lang: Internal Language of the Intelligence Vector Brain

## Summary
Graph-native synthetic language combining Lojban formal grammar + Ithkuil cognitive density + novel graph traversal syntax. Not for human communication — for machine thought.

## Key Features
- 838 Ogden primes as CVC roots (dimension-consonant encoded)
- 6-slot morphological affix system (config, dimension, cardinality, layer, relation)
- Graph traversal operators (@node, #concept, $dim, %layer, ?pattern)
- Deterministic compositionality (primes → composites via rafsi-style fragments)
- Zero ambiguity, trivially parseable, self-describing tokens

## Design Principle
Maximum expressive surface area with minimal tokens. Every token self-describes its dimensional home, abstraction layer, and semantic mass.
