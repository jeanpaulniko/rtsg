# Ogden's Basic English — 850 Word Foundation

## Overview

C.K. Ogden's **Basic English** (1930) reduces the English language to 850 essential words sufficient for international communication. In RTSG, this word set serves as the **linguistic kernel** — the minimal basis for the Linguistic dimension ($d_L$) of the K-Matrix.

These 850 words form a **complete basis** in the sense that any English sentence can be rephrased using only these words plus domain-specific nouns. This is analogous to how a vector space is spanned by its basis vectors — Ogden found the basis of English.

## Connection to RTSG

| RTSG Concept | Ogden Parallel |
|---|---|
| Grothendieck filter | Ogden's 850 is a **linguistic filter** — it projects the full English lexicon onto a 850-dimensional subspace |
| Negative space | The ~170,000 English words *not* in Basic English form the kernel of this filter |
| Spectral decomposition | The 850 words are the top eigenvectors of English by communicative utility |
| K-Matrix $d_L$ | Basic English spans the Linguistic dimension with minimal basis |
| Intelligence equation $U = V/(E \times T)$ | Maximum understanding per word — highest $U$ achievable in language |

## Word Categories

Ogden organized the 850 words into functional classes:

| Category | Count | Role |
|---|---|---|
| Operations (verbs/particles) | 100 | Actions and logical connectors |
| Things (general nouns) | 400 | Objects and concepts |
| Things (picturable nouns) | 200 | Concrete, visualizable objects |
| Qualities (general) | 100 | Properties and attributes |
| Qualities (opposites) | 50 | Paired contrasts |
| **Total** | **850** | |

## RTSG Syntax Extension

Ogden's 850 provides the **lexical substrate**. RTSG extends this with a logical syntax layer inspired by Lojban and formal logic:

### Logical Connectives

| Symbol | Ogden Word | Logical Form | RTSG Role |
|---|---|---|---|
| `∧` | and | Conjunction | Tensor product of concepts |
| `∨` | or | Disjunction | Coproduct / pushout |
| `¬` | not | Negation | Complement / negative space |
| `→` | if...then | Implication | Morphism / causal arrow |
| `↔` | same as | Biconditional | Isomorphism |
| `∀` | every, all | Universal | Sheaf section (global) |
| `∃` | some, any | Existential | Stalk (local existence) |

### Dimensional Tags

Every Ogden word can be tagged with its primary Gardner dimension:

| Tag | Dimension | Example Words |
|---|---|---|
| `[S]` | Spatial | place, distance, direction, position, angle, parallel, square, circle, curve, flat, round, straight |
| `[L]` | Linguistic | word, language, letter, writing, name, reading, say, talk, question, answer, story, prose |
| `[M]` | Logical-Mathematical | number, addition, division, equal, measure, account, balance, comparison, degree, rate, scale, unit |
| `[B]` | Bodily-Kinesthetic | walk, run, jump, push, pull, kick, grip, touch, move, motion, step, stretch |
| `[N]` | Naturalistic | animal, plant, seed, earth, sun, moon, star, river, mountain, rain, wind, weather |
| `[μ]` | Musical (hole) | music, rhythm, song, harmony, bell, sound, voice, noise, thunder, whistle |
| `[I]` | Interpersonal (hole) | friend, brother, sister, family, society, company, committee, nation, government, public |
| `[ι]` | Intrapersonal (hole) | self, mind, belief, feeling, memory, thought, desire, fear, hope, shame, conscious |

### Sentence Structure

RTSG logical syntax follows **Subject–Relation–Object** with explicit typing:

```
[dimension] subject —relation→ object
```

Examples using only Ogden words:

```
[M] number —is_a→ idea
[S] circle —part_of→ form
[N] seed —produces→ plant
[B] foot —enables→ walk
[ι] mind —contains→ thought
[μ] rhythm —relates_to→ music
```

### Filter Notation

Applying a Grothendieck filter to the Ogden basis:

```
F_spatial(850) = {place, distance, direction, position, angle, ...}  (~60 words)
F_logical(850) = {number, addition, equal, measure, ...}             (~45 words)
neg(F_spatial) = 850 \ F_spatial(850)                                (~790 words)
```

The negative space of each dimensional filter contains the words that dimension *cannot see* — its topological silence.
