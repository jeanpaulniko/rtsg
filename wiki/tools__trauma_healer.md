# Trauma Healer App

React application for trauma-informed practitioners.

## Features
- 12-dimension pre/post assessment (0-10 scale per dimension)
- Pre-treatment vs current radar overlay
- Visual graph comparison: collapsed (dashed) vs active (solid) nodes
- Percolation threshold meter (7+ active dims = self-sustaining recovery)
- Six trauma profiles: Sexual Abuse, Physical Violence, Neglect, Emotional Abuse, Accident/Medical, Combat/War
- Each profile maps specific collapsed dimensions with evidence-based healing activities
- Intervention plan generator prioritized by severity
- Language-as-bridge prompts for cross-dimensional reconnection
- Recovery delta bar chart

## Therapeutic Model
- Trauma = collapsed dimensions = disconnected subgraph
- Healing = reconnecting edges = rebuilding the graph
- Percolation threshold = when recovery becomes self-sustaining
- Language = inter-dimensional bus = bridge function

## For Licensed Practitioners Only
This tool supports clinical decision-making. It is not a diagnostic instrument.
