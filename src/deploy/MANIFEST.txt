DATABASE SCHEMA
================================================================================

RTSG DIMENSIONS (3):

Relational Dimension (Basis: 5)
  Coordinates: Symmetry, Connectivity, Centrality, Flow, Clustering
  Gram Matrix: 5x5 correlation matrix (stored in gram_cache)
  Use Case: Network structure analysis

Topological Dimension (Basis: 5)
  Coordinates: Genus, Betti-0, Betti-1, Euler-χ, Boundary
  Gram Matrix: 5x5 correlation matrix (stored in gram_cache)
  Use Case: Shape and topology properties

Geometric Dimension (Basis: 5)
  Coordinates: Curvature, Volume, Area, Geodesic-L, Torsion
  Gram Matrix: 5x5 correlation matrix (stored in gram_cache)
  Use Case: Spatial and curvature properties

INDEXING STRATEGY:

entities:
  - PRIMARY KEY (id)
  - UNIQUE (uuid)
  - INDEX (name, type, created_at)

relations:
  - PRIMARY KEY (id)
  - FOREIGN KEYS (subject_id, object_id) with CASCADE
  - COMPOSITE INDEX (subject_id, predicate, object_id)
  - INDEX (created_at)

dimension_vectors:
  - PRIMARY KEY (id)
  - FOREIGN KEY (entity_id) with CASCADE
  - COMPOSITE INDEX (entity_id, dimension)
  - INDEX (created_at)

gram_cache:
  - UNIQUE (dimension, dim1, dim2, number_system)
  - INDEX (dimension, computed_at)

embeddings:
  - PRIMARY KEY (id)
  - FOREIGN KEY (entity_id) with CASCADE
  - UNIQUE (entity_id, model)
  - IVFFLAT INDEX (vector) with 100 lists

relation_embeddings:
  - PRIMARY KEY (id)
  - FOREIGN KEY (relation_id) with CASCADE
  - UNIQUE (relation_id, model)
  - IVFFLAT INDEX (vector) with 100 lists

TROUBLESHOOTING QUICK LINKS
================================================================================

Service Won't Start:
  1. Check logs: pm2 logs smarthub-api
  2. Verify ports: lsof -i :8080
  3. Check permissions: ls -la /opt/smarthub/bin/

Database Issues:
  1. Test connection: psql -h localhost -U nrte_admin
  2. Check logs: tail -f /var/log/postgresql/postgresql.log
  3. Verify schema: psql -d nrte_production -c "\dt"

TLS Certificate Issues:
  1. View Caddy logs: tail -f /var/log/caddy/smarthub.log
  2. List certificates: caddy list-certstore
  3. Reload config: caddy reload -c /etc/caddy/Caddyfile

Frontend 404 Errors:
  1. Verify build: ls /opt/smarthub/frontend/dist/index.html
  2. Check Caddy routes: caddy validate -c /etc/caddy/Caddyfile
  3. Review web logs: tail -f /var/log/caddy/smarthub.log
