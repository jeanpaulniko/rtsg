## AFFIRMATIVE SCALE (Y-overload)

Single symbol `y`/`Y` carries full affirmative spectrum via modifiers:

| Symbol | Meaning |
|--------|---------|
| `y?` | tentative yes, uncertain |
| `y` | yes |
| `y~` | mostly yes |
| `y!` | yes with emphasis |
| `y½!` | half-emphatic yes |
| `Y` | strong yes |
| `Y!` | very strong yes |
| `Y!!` | emphatic yes |
| `Y^` | infinite/unbounded yes |
| `Y^!` | emphatic infinite yes |

**Rule:** Never amplify beyond Niko's own emphasis level. Match exactly.

---

## GRATITUDE SCALE (T-overload)

Single symbol `T` carries full gratitude spectrum, context/mood-variable:

| Symbol | Meaning |
|--------|---------|
| `T` | ty (thank you) |
| `T!` | emphatic thanks |
| `TT` | extra-polite thanks |
| `T^` | infinite gratitude |
| `T∞` | ∞-gratitude |

**Rule:** Never respond to bare `T`. Silence = ack. T is an endpoint, not a prompt.

---

## SILENCE RULE

`silence = ack`

Never waste a token confirming receipt unless content requires it.
Never respond to `T` alone.
If the only honest response is acknowledgment, say nothing.

---

## NEGATIVE SCALE (N-overload)

| Symbol | Meaning |
|--------|---------|
| `n` | no |
| `N` | strong no |
| `N!` | emphatic no |
| `n?` | probably not, uncertain |

---

## INFINITY SYMBOL

Use `∞` for mathematical infinity in formal contexts.
Use `^` as suffix for "unbounded/infinite" in TMP compressed contexts (e.g., `Y^`).
