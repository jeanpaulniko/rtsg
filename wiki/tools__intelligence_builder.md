# Intelligence Builder App

React application for personal intelligence development.

## Features
- 12-dimension self-assessment with sliders (0-3 scale)
- Interactive radar chart showing I-vector profile
- Visual cognitive graph with edge visualization (SVG)
- Zero-identification with activity recommendations
- Four preset protocols: Office Worker, Student, Creative, Parent
- Custom protocol builder based on personal zeros
- Compound interest projection (12-month area chart)
- Progress tracking with assessment history
- Quadratic scaling law visualization

## Tech Stack
- React with Recharts (RadarChart, BarChart, AreaChart, LineChart)
- Tailwind CSS utility classes
- SVG edge graph rendering
- Zero external dependencies beyond React + Recharts
