# RTSG Scent Protocol — K-Matrix Olfactory Maintenance

## Principle
Olfaction is the oldest sensory system. It bypasses the thalamic relay and projects directly to the hippocampus (memory) and amygdala (emotion). Loss of smell precedes cognitive decline by 5–10 years. A daily scent protocol is a **calibration routine for the deepest brain structures**.

In RTSG terms: each scent activates a specific dimensional tube. A full scent collection keeps all 8 K-Matrix dimensions receiving olfactory input. The slime mold principle applies — don't optimize for one scent. Keep all pathways minimally active. The network self-organizes.

## Implementation

### Physical Setup
- **Morning station**: Small tray near bathroom/kitchen. 8 small dark glass jars with cotton wicks. Refresh oils weekly.
- **Evening station**: Bedside tray. Same format, different scents.
- **Emergency kit**: 5 small roller bottles or inhalers. Carry daily.

### Scent Sources
- Essential oils (therapeutic grade, not synthetic)
- Fresh materials where possible (real coffee grounds, actual pine needles, wet soil after rain)
- Rotate specific varieties within each category to prevent habituation (e.g., alternate Moroccan and Tunisian rosemary)

### Habituation Prevention
The olfactory system adapts quickly to constant stimuli. Counter-measures:
- **Brevity**: 3–5 seconds per scent, not prolonged exposure
- **Variety**: Rotate sources within each dimensional slot
- **Gaps**: Full 24-hour cycle between morning and next morning exposure
- **Contrast**: Alternate warm/cool, sweet/sharp, light/heavy within the sequence

## The Slime Mold Principle
The slime mold doesn't pick one nutrient source. It extends toward all of them simultaneously, then reinforces the strongest connections. The scent protocol works the same way: present all 8 dimensional inputs every day. The brain's network decides which tubes to thicken based on current need. The protocol doesn't prescribe outcomes — it provides the raw material for self-optimization.

No tube starves. No dimension atrophies. The network stays fully connected. That's anti-aging at the sensory level.
